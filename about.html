About This Project

The Air Quality Monitoring System is a web-based platform designed to track and display real-time environmental data such as air pollution levels (PM2.5, PM10, CO2) and weather conditions (temperature, humidity).

By using smart sensors and modern web technologies, this system provides users with accurate and live insights about the air they breatheâ€”helping raise awareness and support better environmental decisions.

Key Features:

  • Live air quality index updates based on location
  • Weather information like temperature & humidity
  • Interactive visualizations (graphs, pie charts)
  • User-friendly interface for all age groups

Technologies Used: HTML, CSS, JavaScript, Sensor APIs, possibly Node.js or Python for backend integration.

The goal of this project is to increase awareness about air pollution and help users make informed decisions about their health and environment.
